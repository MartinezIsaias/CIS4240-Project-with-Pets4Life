# PROJECT – Pet4Life System Proposal	Spring 2025 Semester
**CIS-320 System Analysis and Design** 
<br>
Worked with a team following the Agile workflow to propose a new system for Pet4Life non-profit organization that provides pet and pet owner resources. Planned and designed a new website and system to improve the facilitation of their business processes. Worked on three phases of the SDLC; Analyze, Define, and Design, which were documented as deliverables.
